## switch语句
switch语句是多路分支执行。将一个表达式或是类型跟一系列case进行比较，匹配的分支会被执行。

Go中的switch是我见过的语言中最灵活的，不像有些语言只能是数字或者byte可以。

```go
switch week {
    case "Mon":
        ...
    case "Thu":
        ...
    default:  //如果没有找到合适的匹配，执行default
        ...
}

switch i {
    case 3:
        ...
    case 4,5,6: //相当于if i==4 || i==5 || i==6
        ...
    default:  
        ...
}

//GO中还可以使用赋值语句

switch i := yourfun(); {
    case i < 0:
        ...
    case i > 0:
        ...
    default:
        // 0
}
```
