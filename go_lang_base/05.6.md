## select语句
select语句从多个发送和接收操作中选择一个执行。它看上去像是switch语句，只不过它的每一个case都是通信操作。

```go
select {
case u:= <- ch1:
        ...
case v:= <- ch2:
        ...
        ...
default: // no value ready to be received
        ...
}
```
