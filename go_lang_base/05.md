## 基本语句
在上面两章我们介绍了GO的基本类型、标识符、函数和方法，这些都是编写程序的基础。在本章，我们将讲述GO的流程语句。有了这些，你就可以写出完整的程序了。

GO有以下几种程序流程结构：

1. if-else 结构
2. switch 结构
3. for (range) 结构 GO只有一个for结构，没有while结构。
4. select 结构，用于 channel 的选择

#### Go 完全省略了 if、switch 和 for 结构中条件语句两侧的括号。这点大家要注意。
